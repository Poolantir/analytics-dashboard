# Poolantir

Analytics dashboard for pool monitoring.

## Pages

- **Monitoring** — Real-time bento grid overview of pool metrics
- **Bathrooms** — Building restroom availability with floor plans
- **Settings** — User preferences and session management

## Stack

- [Plotly Dash](https://dash.plotly.com/)
- [Dash Mantine Components](https://www.dash-mantine-components.com/)
- [Dash Iconify](https://github.com/snehilvj/dash-iconify)

## Run

```bash
docker compose up --build
```

The dashboard is served at `http://localhost:8000`.
